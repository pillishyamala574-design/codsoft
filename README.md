# Rule-based chatbot
## description:
this task is a simple rule-based chatbot developed using python, the chatbot responds to user input.

## features:
responds to greetings and common questions
Uses predefined rules for conversation
easy to understand and modify
beginner-friendlu AI project.

## technologies used:
python

## how it work:
the chatbot matches user input with predefined keywords and returns the corresponding response. if no matching rule is found,it provide a default response.

## example:
user: hello
bot: hi! how can i help you?
user: what is your name?
bot: I am a rule based chatbot.
user:bye
bot: goodbye! have a nice day.

## how to run:
install python
download the project files 
open the terminal in the project folder
run the command:
python chatbo.py
