# Movie Recommendation System
This is a simple Python project that recommends movies based on user preferences.

## About the Project
This system suggests movies based on selected genre or category. It uses a basic filtering approach using Python lists and dictionaries.

## Features
- Genre-based movie recommendations  
- Simple and fast logic  
- User input-based suggestions  
- Console-based application  

## How to Run
python movie.py

## Technologies Used
Python 
Lists and Dictionaries
Conditional statements
Basic programming logic

## Learning Outcomes
Understanding recommendation system basics
Working with Python data structures
Improving logical thinking
Handling user input and output
