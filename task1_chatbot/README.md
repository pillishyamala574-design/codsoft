# Rule-Based Chatbot

## Description
This project is a simple Rule-Based Chatbot developed using Python. The chatbot responds to user inputs based on predefined rules and patterns. It can answer basic questions, greet users, and carry out simple conversations.

## Features
- Responds to greetings and common questions
- Uses predefined rules for conversation
- Easy to understand and modify
- Console-based interactive chatbot
- Beginner-friendly AI project

## Technologies Used
- Python

## How It Works
The chatbot matches user input with predefined keywords and returns the corresponding response. If no matching rule is found, it provides a default response.

## Example Conversation

User: Hello

Bot: Hi! How can I help you today?

User: What is your name?

Bot: I am a Rule-Based Chatbot.

User: Bye

Bot: Goodbye! Have a nice day.

## How to Run

1. Install Python.
2. Download the project files.
3. Open the terminal in the project folder.
4. Run the command:

```bash
python chatbot.py
```
