#  Tic-Tac-Toe AI
This is a simple Python project where a human player plays Tic-Tac-Toe against an AI opponent. The AI uses the Minimax algorithm to make optimal moves.

##  About the Project
Tic-Tac-Toe is a classic game played on a 3x3 grid. In this project, the user plays against an AI that is designed to never lose if played correctly.

##  Features
- Human vs AI gameplay  
- AI uses Minimax algorithm  
- Smart decision making  
- Console-based interface  
- Simple and interactive game  

##  AI Logic
The AI uses the **Minimax Algorithm**:
- It checks all possible moves
- Predicts future game outcomes
- Assigns scores:
  - Win → +1  
  - Lose → -1  
  - Draw → 0  
- Chooses the best possible move
